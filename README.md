# OOTD iOS App

Swift 기반으로 개발한 OOTD(Outfit Of The Day) 공유 애플리케이션입니다.

사용자가 자신의 코디를 공유하고 다른 사용자들의 스타일을 피드 형태로 확인할 수 있도록 설계한 iOS 앱 프로젝트입니다.

---

## 📱 프로젝트 소개

패션 코디를 기록하고 공유할 수 있는 모바일 애플리케이션을 목표로 제작한 프로젝트입니다.  
SwiftUI 기반으로 화면을 구성하고, 다양한 View를 활용하여 iOS 앱 구조를 설계했습니다.

---

## ✨ 주요 기능

- 로그인 화면 구현
- 사용자 피드 기능
- 채팅 기능
- 마켓 기능
- 옷장 관리 기능
- 코디 추천 기능
