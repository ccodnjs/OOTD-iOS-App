🧩 Project Overview

OnFit은 패션 스타일을 기록하고 공유할 수 있는 SNS 기반 패션 플랫폼입니다.

사용자는 자신의 옷을 디지털 옷장에 등록하고
OOTD 게시글을 통해 스타일을 공유하며
안 입는 옷은 마켓에서 거래할 수 있습니다.

🎯 Target Users

자신의 스타일을 기록하고 공유하고 싶은 사용자

패션에 관심 있는 10~20대 사용자

중고 의류 거래 및 합리적인 소비에 관심 있는 사용자
